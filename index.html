TONE ORACLE — The Devil

A real-time random oscillator engine that generates unpredictable tones using 15 different voice types. Each event has a randomly chosen frequency (20–7000 Hz, log-uniform distribution), duration (40–2000 ms), amplitude, and stereo pan. No two moments sound the same.

15 Voice Types: Sine, Sawtooth (2× oversampled with polyBLEP anti-aliasing), Square, Pulse (variable duty cycle via PeriodicWave), Triangle, White Noise, Pink Noise (Paul Kellet filter), Filtered Noise Burst (resonant bandpass), FM Synthesis (carrier + modulator with random ratio and index), Ring Modulation, Exponential Chirp, Percussive Click, Detuned Stack (3–5 beating sines), Glitch (gated at random rate), Tremolo Sine.

  • Mode 1 — Pure Sine: One clean sine tone at a time, long meditative gaps.
  • Mode 2 — Noise Machine: Random voice type per event. The full variety.
  • Mode 3 — All Together: 4 parallel voice slots, each independently scheduling random voices.
  • Mode 4 — Cacophony: 6 parallel slots with heavy-tailed gap distribution (bursts, long silences, maximum chaos).

BABEL ORACLE — The Neighbor

Generates phonotactically valid nonsense words in 22 languages using language-specific syllable inventories (onset consonants + vowels + codas). Each language has its own connector words for natural sentence flow. Speech is produced via the browser's SpeechSynthesis API with automatic voice discovery — the correct accent and prosody for each language comes free.

  • Mode 1 — Words: Single words, each in a random language, long gaps. Meditative.
  • Mode 2 — Sentences: Full sentences (3–8 words), each in a random language.
  • Mode 3 — Babel: Rapid-fire words with very short gaps. Tower of Babel.
  • Mode 4 — Podcast: Two persistent hosts with fixed voices alternate sentences. Re-rolled every 8 exchanges.
  • Mode 5 — City: 11 urban sound generators — cars, horns, sirens, hammering, dogs, birds, footsteps, wind, doors, crowd chatter, trains.
  • Mode 6 — Chaos: Babel speech + City sounds combined.
  • Mode 7 — Robot: Formant-synthesized speech routed through the FX chain. 3-pole biquad bandpass resonators per phoneme. All effects process the voice directly.

22 Languages: English, Spanish, French, German, Italian, Portuguese, Russian, Japanese, Chinese, Korean, Arabic, Hindi, Turkish, Dutch, Swedish, Polish, Hebrew, Thai, Vietnamese, Indonesian, Greek, Czech.

DEPTH SIGNAL — The Spy (Limited Edition)

13 procedural signal generators inspired by submarine communications, marine life, and deep-sea ambience. All sounds are pre-generated into a buffer pool at initialization for glitch-free real-time playback — no main-thread blocking during performance.

  • Morse: Random dit/dah beeps at random pitch (400–1200 Hz).
  • Radio: Shortwave static + tuning sweeps + crackling.
  • Electrical: Transformer hum, relay clicks, spark crackle, capacitor whine.
  • Sonar: Classic submarine pings with exponential decay and faint echo returns.
  • Dolphin: Click trains (5–15 clicks accelerating) and whistles (sine sweep 4–18 kHz).
  • Owl: Low hoots with pitch drop, screech owl calls, cricket chirps.
  • Deep: All types mixed (4 concurrent slots).
  • Abyss: Ultra-low drones (20–60 Hz), metallic groans (FM synthesis), pressure creaks, whale calls (slow sine glide 40–200 Hz over 5–10 seconds).
  • Sweep: Ultra-random frequency sweeps: 2–20 seconds, any waveform (sine/saw/square/triangle/noise-modulated), any direction, any envelope.
  • Signal: Morse + Sonar cacophonic conversation (4 slots).
  • All: Every generator simultaneously (5 slots). Maximum submarine pandemonium.

SHARED FX ENGINE

All three oracles share the same professional-grade effects chain, built with the Web Audio API:

REVERB
Convolution reverb with 2-second synthetic impulse response. Anti-alias lowpass at 9.5 kHz on the wet return.
DELAY
350 ms feedback delay with tape-style darkening (6.5 kHz lowpass inside the feedback loop).
8-BIT
Bitcrusher via AudioWorklet — reduces bit depth (16→3) and sample rate simultaneously. WaveShaper fallback for older browsers.
HI CUT / LO CUT
BiquadFilter EQ. Lowpass 200–20000 Hz, Highpass 20–2000 Hz.
SOFT CLIPPER
WaveShaper with 4× oversampling. Transparent below ±0.7, smooth tanh saturation above.
RECORDING
WAV (16-bit PCM via ScriptProcessor) or WEBM (MediaRecorder). Auto-download on stop.

BACKGROUND NOISE

8 switchable ambient noise types, each with a per-type anti-alias filter:

  • Brown: Warm low rumble (lowpass 200 Hz).
  • Pink: Balanced 1/f spectrum.
  • Rain: Bandpass-filtered noise with random drops.
  • Wind: Slowly modulated low-frequency swell.
  • Street: Low rumble + horn pings + distant hiss.
  • Night: Soft noise + dual cricket chirps (4.2 / 3.8 kHz).
  • Hum: 50 Hz mains + harmonics.
  • Off: Complete silence.

Plus a persistent 60 Hz drone with 0.3 Hz LFO tremolo, independently controllable.

TECHNICAL NOTES

  • 100% client-side — zero server dependencies, runs from a single HTML file.
  • Web Audio API for all synthesis and effects processing.
  • Anti-aliasing: polyBLEP on saw/square/pulse (Tone Oracle), 2× oversampling + FIR lowpass, AA lowpass on all three FX wet returns, 4× WaveShaper oversample on the soft clipper.
  • Buffer pool system (Depth Signal) pre-generates audio for glitch-free playback.
  • Responsive design — works on desktop and mobile browsers.
  • Works offline from local files.
